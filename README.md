### WordQuest

A simple yet engaging **Python-based word guessing game** featuring multiple difficulty levels and colorful console output using **Colorama**.

### Features
- Interactive console gameplay  
- Colored output for a fun experience  
- Three difficulty levels (Easy, Medium, Hard)  
- Input validation and smart feedback  
- External word list support (`words.txt`)
